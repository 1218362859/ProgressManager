## Listen the progress of  downloading and uploading in Okhttp (compatible Retrofit and Glide).

ProgressManager 一行代码即可监听 App 中所有网络链接的上传以及下载进度,包括 Glide(需要将下载引擎切换为 Okhttp)的图片加载进度
